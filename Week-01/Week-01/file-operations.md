# File Operations

## Commands Practiced

```bash id="px2jlwm"
touch
mkdir
cp
mv
rm
```

## Concepts Learned

* `touch` creates empty files
* `mkdir` creates directories
* `cp` copies files/directories
* `mv` moves or renames files
* `rm` removes files

## Key Learnings

* `mv` handles both moving and renaming
* Commands operate relative to current directory
* Files can be manipulated using paths
