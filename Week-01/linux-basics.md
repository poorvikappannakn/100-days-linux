# Linux Basics

## Filesystem

Linux uses a hierarchical filesystem structure.

## Commands Practiced

```bash
pwd
ls
ls -l
ls -a
ls -la
cd
cd ..
cd ~
```

## Concepts Learned

* Current working directory
* Hidden files
* Relative vs absolute path
* Detailed file listing using `ls -l`

## Key Learnings

* Hidden files start with `.`
* Relative paths depend on current location
* Absolute paths start from `/`
* `ls -l` shows permissions and file details
